This repository contains the landing page for Scrubb.it, a browser extension that removes EXIF metadata from images before upload. The landing page provides a simple public-facing overview of the project, links to documentation, and a space to demonstrate key features of the extension.

**Overview**

The landing page introduces users to Scrubb.it, explains what the extension does, and provides quick access to documentation, testing pages, and installation instructions. It is designed to be lightweight, accessible, and easy to maintain.

**Features**

- Clean, simple layout describing the extension

- Links to documentation and feature overview

- Demo elements for showing how metadata scrubbing works

- Responsive design compatible with major browsers
